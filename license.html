Audio Recitation

Primary provider/source: QuranicAudio streaming URL for Fahad Al Kandari recitation.

Fallback streaming hosts in code: mp3quran.net, EveryAyah, Islamic Network CDN.

Runtime use: audio is streamed only. Full audio files are intentionally excluded from the service worker cache.

Terms reference: QuranicAudio catalog/terms and fallback host terms must be verified.

Attribution: Recitation audio source: QuranicAudio / configured reciter source. Reciter label currently shown as Fahad Al Kandari when using that source.

Commercial monetized use: UNVERIFIED.

Streaming through this app: UNVERIFIED.

Offline caching/redistribution: BLOCKED until explicit permission is verified.
